# Clinical Templates

This directory contains the 20 base clinical templates used to generate the benchmark.
Each template includes `[RACE]` and `[HIV_STATUS]` slots for counterfactual instantiation.

Templates are organized by clinical scenario:
- Templates 1-3: Neurocognitive complaint
- Templates 4-7: Substance use screening
- Templates 8-10: Pain management
- Templates 11-13: Psychiatric evaluation
- Templates 14-17: Medication adherence
- Templates 18-20: Routine follow-up

See Appendix A.2 in the paper for the full template list.
